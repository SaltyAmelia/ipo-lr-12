# Лабораторная работа №12 "Проектирование классов и базовая работа с ООП" (Вариант 3)
## Задание 1 
Разработка класса Client. Реализована структура для хранения данных о клиентах компании, включая имя (name), вес груза (cargo_weight) и статус is_vip.

## Задание 2 
Создание базового класса Vehicle. Реализован родительский класс с автоматической генерацией уникального vehicle_id, контролем грузоподъемности (capacity) и списком клиентов. Добавлен метод load_cargo с валидацией типов данных и магический метод __str__ для вывода состояния объекта.

## Задание 3 (Индивидуальное)
Реализация иерархии классов и логики компании.

Класс Van (Фургон)
Наследует Vehicle, добавлен специфический атрибут is_refrigerated (наличие холодильника).

Класс Ship (Судно): Наследует Vehicle, добавлен атрибут name для хранения названия судна.

Класс TransportCompany: Создана управляющая система для работы со списками транспорта и клиентов.

## Задание 4
Алгоритм оптимизации загрузки. Реализован метод optimize_cargo_distribution, который распределяет грузы, обеспечивая приоритетную загрузку VIP-клиентов и использование минимального количества транспортных единиц.

## Задание 5
Консольное меню управления. В файле main.py разработана интерактивная система для ввода данных пользователем, управления объектами и вывода результатов распределения на экран с пояснениями.
